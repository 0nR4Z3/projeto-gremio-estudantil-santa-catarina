# 🌟 União PESC — E.E.E.M. Santa Catarina

> Website institucional do Grêmio Estudantil União PESC da E.E.E.M. Santa Catarina, Caxias do Sul/RS.

🔗 **Live Demo:** [https://seu-usuario.github.io/uniao-pesc/](https://seu-usuario.github.io/uniao-pesc/)  
📧 **Contato:** sta.catarina@yahoo.com.br | 📸 [@eeem_santacatarina](https://instagram.com/eeem_santacatarina)

---

## ✨ Destaques

- 🎬 Intro cinematográfica com estrela 3D (Three.js) e suporte a vídeo de fundo
- 🌗 Design system coeso: tokens, grid fluido, animações GSAP + ScrollTrigger
- 📱 100% responsivo, mobile-first, com menu fullscreen otimizado
- ♿ Acessibilidade: `prefers-reduced-motion`, `aria-labels`, cursor fallback
- ⚡ Performance: lazy loading, IntersectionObserver, animações pausadas fora da viewport
- 📄 Multi-página: `index.html`, `historia.html`, `atividades.html`, `gremio.html`, `matriculas.html`, `contato.html`

## 🛠️ Tecnologias

| Camada        | Ferramentas                          |
|---------------|--------------------------------------|
| Estrutura     | HTML5 semântico                      |
| Estilização   | CSS3 + Custom Properties + Clamp     |
| Animações     | GSAP 3 + ScrollTrigger + ScrollTo    |
| 3D / Canvas   | Three.js r128                        |
| Deploy        | GitHub Pages / Vercel / Netlify      |
| Fontes        | Bebas Neue, Cormorant Garamond, Space Mono |
